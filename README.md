# App License Server

This repo hosts the application license status for IPSSS backend.

## Usage

- **URL**: `https://sudhanshu25012002.github.io/app-license-server/status.json`
- **Returns**: `{ "status": true }` when app should run
- **Kill app**: Change to `{ "status": false }` and push

## Important

⚠️ Do not delete this repository - it controls the application's operational status.
